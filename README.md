### 서비스 내용 및 기능
넘쳐나는 취미 플랫폼 클래스들을 한 번에 모아볼 수 있는 사이트
- 유명 플랫폼 7개사 클래스 인기, 가격 한 번에 비교 
- 구글/네이버/카톡 아이디로 간편하게 소셜 로그인 
- 사용자별 관심 지역, 분야 설정 가능
